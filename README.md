## Рюе **рЃњрЃљрЃџрЃљрЃЦрЃАрЃБрЃАрЃў**: рЃАрЃћрЃарЃЋрЃўрЃА рЃЏрЃўрЃгрЃЮрЃЊрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЊрЃљрЃАрЃљрЃЦрЃЏрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃљрЃ«рЃљрЃџрЃў рЃћрЃарЃљ ­ЪњФ

**рЃњрЃљрЃџрЃљрЃЦрЃАрЃБрЃАрЃў** рЃљрЃарЃўрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃћрЃарЃЌрЃћрЃЉрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃЏрЃљрЃФрЃўрЃћрЃЉрЃџрЃћрЃЉрЃАрЃљ рЃЊрЃљ рЃЎрЃЋрЃљрЃџрЃўрЃцрЃўрЃфрЃўрЃБрЃа рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃћрЃЉрЃА.

­Ъїљ **рЃЕрЃЋрЃћрЃю рЃЋрЃфрЃЋрЃџрЃўрЃЌ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃћрЃЉрЃљрЃА** - рЃћрЃарЃЌрЃ»рЃћрЃарЃљрЃЊрЃў рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃўрЃЊрЃљрЃю рЃњрЃарЃФрЃћрЃџрЃЋрЃљрЃЊрЃўрЃљрЃю рЃЌрЃљрЃюрЃљрЃЏрЃерЃарЃЮрЃЏрЃџрЃЮрЃЉрЃљрЃЏрЃЊрЃћ.  

­ЪњА **рЃЕрЃЋрЃћрЃюрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ** рЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃА, рЃ«рЃЮрЃџрЃЮ рЃцрЃўрЃюрЃљрЃюрЃАрЃБрЃа рЃерЃћрЃЌрЃљрЃюрЃ«рЃЏрЃћрЃЉрЃћрЃЉрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃў рЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃЉрЃћрЃю.

## ­ЪћЦ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃў  

### ­ЪцЮ **рЃЏрЃДрЃўрЃАрЃўрЃћрЃарЃў рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ**  
- рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃерЃћрЃЌрЃљрЃюрЃ«рЃЏрЃћрЃЉрЃћрЃЉрЃў рЃАрЃљрЃбрЃћрЃюрЃЊрЃћрЃарЃЮ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃЌ  
- рЃњрЃџрЃБрЃЋрЃў рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃљ рЃърЃўрЃарЃљрЃЊ рЃЏрЃћрЃАрЃўрЃ»рЃўрЃюрЃњрЃќрЃћ  
- рЃАрЃгрЃарЃљрЃцрЃў рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃљрЃарЃ«рЃћрЃЉрЃў  

### ­Ъћњ **рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃЏрЃћрЃАрЃўрЃ»рЃўрЃюрЃњрЃў**  
- рЃЊрЃљрЃерЃўрЃцрЃарЃБрЃџрЃў рЃЕрЃљрЃЌрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА  
- рЃЋрЃљрЃЊрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃ«рЃЊрЃўрЃА рЃърЃўрЃарЃЮрЃЉрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃ«рЃўрЃџрЃЋрЃљ  
- рЃЏрЃБрЃЊрЃЏрЃўрЃЋрЃў рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃЎрЃџрЃўрЃћрЃюрЃбрЃћрЃЉрЃЌрЃљрЃю  

### ­Ъћё **рЃЮрЃарЃЏрЃљрЃњрЃў рЃарЃЮрЃџрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ**  
- рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃњрЃљрЃЊрЃљрЃарЃЌрЃЋрЃљ рЃЏрЃўрЃЏрЃгрЃЮрЃЊрЃћрЃЉрЃџрЃўрЃА/рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃарЃЮрЃџрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА  
- рЃЏрЃарЃљрЃЋрЃљрЃџрЃцрЃћрЃарЃЮрЃЋрЃљрЃюрЃў, рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃАрЃљрЃќрЃЮрЃњрЃљрЃЊрЃЮрЃћрЃЉрЃљ  
- рЃЏрЃћрЃбрЃў рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃ«рЃћрЃџрЃЌ  

### ­Ъњ░ **рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў рЃњрЃљрЃЊрЃљрЃ«рЃЊрЃћрЃЉрЃќрЃћ**  
Рџа№ИЈ **рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў**: рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ рЃљрЃа рЃљрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃћрЃЉрЃА рЃцрЃўрЃюрЃљрЃюрЃАрЃБрЃа рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃА  
- рЃДрЃЋрЃћрЃџрЃљ рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃљ рЃ«рЃЊрЃћрЃЉрЃљ рЃБрЃерЃБрЃљрЃџрЃЮрЃЊ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА  
- рЃАрЃарЃБрЃџрЃў рЃЌрЃљрЃЋрЃўрЃАрЃБрЃцрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЊрЃљрЃ«рЃЊрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃљрЃерЃў
